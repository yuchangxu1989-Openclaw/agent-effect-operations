# AEO（Agent Effect Operations）

Agent 效果运营平台。AEO 自动发现 AI agent 效果漂移、诊断根因、运行数据闭环工具链，给出可落地的效果优化方案。

---

Automated agent performance monitoring and optimization platform. AEO detects effect drift in AI agents, diagnoses root causes, runs data-driven feedback loops, and delivers actionable optimization plans.

## 愿景

AI agent 的效果退化是无声的。Prompt 漂移、工具输出偏移、用户预期演变——但目前没有系统化的方式来检测、诊断和修复这些回退。AEO 填补这个空白。

AEO 位于你的 agent 集群和质量标准之间。它持续监控 agent 输出与基线预期的偏差，在用户察觉之前标记异常，并编排从发现到修复的完整反馈闭环。

---

## Vision

AI agents degrade silently. Prompts drift, tool outputs shift, user expectations evolve — but there's no systematic way to detect, diagnose, and fix these regressions. AEO fills that gap.

AEO sits between your agent fleet and your quality standards. It continuously monitors agent outputs against baseline expectations, flags deviations before users notice, and orchestrates the feedback loop from detection to resolution.

## 规划中的能力

- **漂移检测** — 持续监控 agent 输出质量，与版本化基线对比
- **根因诊断** — 自动分析效果退化与 prompt 变更、工具更新或数据偏移之间的关联
- **反馈闭环编排** — 从异常发现到数据采集、实验设计、修复部署的端到端流水线
- **优化建议** — 可落地的优化方案，附带预期影响评估和回滚策略

---

## Planned Capabilities

- **Drift detection** — continuous monitoring of agent output quality against versioned baselines
- **Root cause diagnosis** — automated analysis linking effect degradation to prompt changes, tool updates, or data shifts
- **Feedback loop orchestration** — end-to-end pipeline from anomaly detection through data collection, experiment design, and fix deployment
- **Optimization recommendations** — actionable plans with expected impact estimates and rollback strategies
